*向浏览器警告窗口输出内容*

```html
alert("hello world");
```

*向<body>中输出语句*

```html
document.write("hello world");
```

*向控制台输出语句*

```html
console.log("hello world");
```

:up:如果三个语句同时写在==<script></script>==中则会按照自上而下的顺序执行。
