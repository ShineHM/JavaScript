---

:black_medium_small_square: Null（空值）类型的值==只有一个==就是Null

null这个值专门用来表示一个为空的对象

```
var a = null；
console.log(a);
```

控制台会输出null

使用typeof检查类型，则会返回object

当声明一个变量但不给它赋值时，他的值就是undefined

```
var b;
console.log(b):
```

控制台会输出undefined

使用typeof检查一个undefined时会返回undefined
